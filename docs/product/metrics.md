# Product Metrics

## Activation

- Completed language selection
- Completed tax profile
- Added first income source
- Added first expense
- Asked first assistant question

## Engagement

- Weekly active users
- Transactions added per active user
- Documents uploaded per active user
- Assistant questions per active user
- Export packs generated

## Trust and quality

- Percentage of assistant answers with source references
- Percentage of high-risk answers escalated
- User-confirmed vs rejected AI extraction suggestions
- Tax estimate recalculation errors
- Support tickets per active user

## Business

- Free-to-paid conversion
- Annual subscription conversion
- Accountant pack purchases
- Churn
